# 04. 상품 소싱과 검증 기준

## 1. 상품 소싱의 기본 관점

부업형 구매대행에서 상품 소싱은 “많이 찾기”보다 “잘 걸러내기”가 중요하다.

초기에는 다음 조건을 만족하는 상품만 후보로 둔다.

- 작고 가볍다.
- 파손 가능성이 낮다.
- 규제가 적다.
- 옵션이 단순하다.
- 브랜드·상표권 리스크가 낮다.
- 반품 사유가 적다.
- 국내 소비자가 구매대행을 감수할 이유가 있다.
- 상세페이지 작성에 필요한 정보가 충분하다.

## 2. 초기 제외 권장 카테고리

초기에는 다음 카테고리를 제외하는 것이 안전하다.

| 카테고리 | 제외 이유 |
|---|---|
| 식품·건강기능식품 | 식약처, 성분, 반입차단, 표시광고 리스크 |
| 화장품 | 성분, 기능성 표현, 수입/판매 규제 가능성 |
| 의료기기 | 인허가·표시광고 리스크 |
| 전기·전자제품 | KC, 플러그, 전압, 배터리, 안전인증 이슈 |
| 어린이제품 | 어린이제품 안전 특별법 관련 리스크 |
| 유명 브랜드 상품 | 가품·상표권·병행수입 오해 리스크 |
| 고가 상품 | 환불·분쟁·관부가세 부담 큼 |
| 대형·중량 상품 | 배송비·파손·반품비 부담 큼 |
| 유리·세라믹 등 파손품 | 파손 CS 가능성 큼 |
| 액체·분말·자석·칼날 포함 품목 | 배송 제한 가능성 |

## 3. 초기 적합 후보

초기에는 다음 유형이 상대적으로 적합하다.

- 비전기 생활잡화
- 소형 정리용품
- 취미용 소모품
- 반려동물 비식품 소품
- 사무용 소품
- 차량용 비전기 액세서리
- 캠핑·아웃도어 소형 액세서리
- 패션잡화 중 브랜드 리스크가 낮은 품목
- 옵션이 단순한 인테리어 소품

단, “상대적으로 적합”하다는 뜻이지 무조건 안전하다는 뜻은 아니다. 각 상품별로 규제·통관·상표권·배송 조건을 확인해야 한다.

## 4. 상품 후보 검증 기준

### 4.1 수요 검증

확인 항목:

- 국내 오픈마켓에 유사 상품이 있는가
- 리뷰가 충분히 있는가
- 최근에도 판매되는가
- 검색 키워드가 명확한가
- 고객이 어떤 문제를 해결하려고 사는가
- 구매대행 배송기간을 감수할 만큼 차별점이 있는가

### 4.2 공급 검증

확인 항목:

- 해외 판매자의 평점과 리뷰
- 상품 리뷰 수
- 옵션 재고 안정성
- 원본 이미지와 상세정보 품질
- 가격 변동 빈도
- 배송 가능 여부
- 배대지 주소 사용 가능 여부

### 4.3 경쟁 검증

확인 항목:

- 국내 최저가
- 국내 평균 판매가
- 리뷰 많은 판매자의 상세페이지 수준
- 배송기간 비교
- 옵션 구성 비교
- 국내 재고 판매자 존재 여부
- 가격 외 차별화 가능성

### 4.4 손익 검증

확인 항목:

- 모든 비용 반영 후 마진이 남는가
- 환율 3~5% 변동에도 손실이 나지 않는가
- 반품 1건 발생 시 누적 이익이 사라지지 않는가
- 플랫폼 수수료 반영 후 가격 경쟁력이 있는가
- 관부가세 발생 가능성이 고객에게 명확히 고지되는가

## 5. 상품 검증 점수표

| 항목 | 점수 기준 |
|---|---|
| 규제 리스크 | 낮을수록 고점 |
| 배송 리스크 | 작고 가벼울수록 고점 |
| 파손 리스크 | 파손 가능성이 낮을수록 고점 |
| 옵션 복잡도 | 단순할수록 고점 |
| 국내 수요 | 유사 상품 판매 흔적이 있을수록 고점 |
| 경쟁 강도 | 가격 경쟁이 과도하지 않을수록 고점 |
| 마진 | 비용 반영 후 충분할수록 고점 |
| 상세페이지 작성 가능성 | 정보와 이미지가 충분할수록 고점 |
| CS 리스크 | 사용법·사이즈·호환성 문제가 적을수록 고점 |

권장 판정:

```text
80점 이상: 테스트 후보
60~79점: 조건부 후보
40~59점: 보류
40점 미만: 제외
```

## 6. P0 서비스에 반영할 소싱 기능

상세페이지 생성 AI 서비스의 P0에는 다음 기능을 우선 고려한다.

- 상품 URL 입력
- 상품명/가격/옵션/이미지 정보 추출
- 금지·주의 카테고리 체크리스트
- 원가·마진 계산 템플릿
- 국내 경쟁 상품 메모 입력
- 상품 리스크 등급 산정
- 상세페이지 생성 가능 여부 판정
- 등록 전 누락 항목 체크
