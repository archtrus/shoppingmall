# 02. 부업형 해외 구매대행 사업 모델

## 1. 기본 운영 철학

부업형 구매대행의 목표는 단기간 고매출이 아니라, **주업을 해치지 않으면서 지속 가능한 운영 구조**를 만드는 것이다.

따라서 초기 전략은 다음과 같아야 한다.

- 작게 시작한다.
- 고위험 품목을 피한다.
- 판매채널을 늘리지 않는다.
- 상품 수보다 운영 안정성을 우선한다.
- 주문당 손익을 반드시 기록한다.
- CS가 복잡한 상품은 제외한다.
- 자동화는 운영 SOP가 정리된 뒤 적용한다.

## 2. 권장 초기 모델

| 항목 | 권장안 |
|---|---|
| 운영 형태 | 무재고 주문형 구매대행 |
| 국가/권역 | 1개 국가 또는 1개 권역부터 시작 |
| 판매채널 | 국내 오픈마켓 1개부터 시작 |
| 소싱처 | 1~2개 플랫폼으로 제한 |
| 초기 SKU | 30~50개 테스트 |
| 카테고리 | 비전기·비식품·비화장품·비의료기기·비아동·비브랜드 민감 품목 |
| 주문 확인 | 하루 2회 이상 |
| 기록 도구 | Google Sheets / Excel |
| 자동화 수준 | 반자동 체크리스트 + 템플릿 중심 |

## 3. 부업 관점의 시간 설계

가용 시간이 적은 경우, 주간 업무는 다음처럼 나누는 것이 현실적이다.

### 보수형: 주 5~8시간

| 업무 | 주간 시간 |
|---|---:|
| 상품 후보 조사 | 1.5~2시간 |
| 상품 검증·마진 계산 | 1~1.5시간 |
| 상품 등록·상세페이지 작성 | 1.5~2시간 |
| 주문·배송·CS 확인 | 매일 10~20분 |
| 정산·기록·개선 | 30분~1시간 |

### 확장형: 주 8~12시간

| 업무 | 주간 시간 |
|---|---:|
| 상품 후보 조사 | 2~3시간 |
| 상품 검증·마진 계산 | 1.5~2시간 |
| 상품 등록·상세페이지 작성 | 2~3시간 |
| 주문·배송·CS 확인 | 매일 20~30분 |
| 성과 분석·개선 | 1~1.5시간 |

## 4. 부업 운영에서 피해야 할 착각

### 4.1 상품 수가 많으면 자연히 팔린다는 착각

구매대행 시장에는 대량 등록 전략이 존재하지만, 개인 부업자가 처음부터 따라 하기에는 위험하다.

대량 등록은 다음 문제를 만든다.

- 품절 관리 불가
- 가격 변동 대응 불가
- 금지 품목 혼입
- 상세페이지 품질 저하
- CS 폭증
- 반품 조건 미숙지
- 플랫폼 제재 리스크

초기에는 상품 수보다 **상품 검증 품질과 운영 재현성**이 중요하다.

### 4.2 해외 가격이 싸면 팔린다는 착각

국내 판매가는 다음 비용을 모두 포함해야 한다.

- 해외 상품가
- 해외 현지 배송비
- 배대지 비용
- 국제배송비
- 국내배송비
- 플랫폼 수수료
- 결제 수수료
- 환율 변동
- 관부가세 가능성
- 반품·분실 리스크 비용
- 판매자 마진

따라서 해외 가격이 싸 보여도 최종 판매가는 경쟁력이 없을 수 있다.

### 4.3 오픈마켓에 올리면 책임이 줄어든다는 착각

오픈마켓에서 판매하더라도 고객 대응, 배송 지연, 환불, 반품, 상품정보 고지, 세무 기록은 판매자 책임이다.

플랫폼은 판매 기회를 제공할 뿐, 운영 리스크를 대신 떠안지 않는다.

## 5. 수익 구조

구매대행 사업의 실질 수익은 다음과 같이 계산해야 한다.

```text
판매가
- 해외 상품가
- 해외 현지 배송비
- 배대지 비용
- 국제배송비
- 국내배송비
- 플랫폼 수수료
- 결제 수수료
- 환율 버퍼
- 예상 CS/반품 리스크 비용
= 대행수수료 및 판매자 이익
```

세무상 처리에서는 물품대금과 배송비를 매출로 볼지, 대행수수료만 매출로 볼지에 대한 요건과 증빙 관리가 매우 중요하다. 사업 초기부터 주문별 증빙을 보관해야 한다.

## 6. 초기 성공 기준

초기 1~2개월은 매출보다 다음 지표를 성공 기준으로 삼는다.

- 상품 등록 프로세스를 반복 가능하게 만들었는가
- 금지·고위험 품목을 걸러낼 수 있는가
- 주문 1건을 끝까지 처리해봤는가
- 배송 지연 CS를 대응해봤는가
- 주문별 원가·마진 기록이 남는가
- 반품 발생 시 손익 영향을 계산할 수 있는가
- 상세페이지 생성 AI에 필요한 입력값이 정의되었는가
