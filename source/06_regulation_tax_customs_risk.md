# 06. 법규·세무·통관·리스크 관리

## 1. 리스크 관리 원칙

해외 구매대행은 판매자가 재고를 보유하지 않더라도 고객에게 상품을 판매하고, 주문·배송·환불을 관리하는 사업이다.

따라서 다음 리스크를 처음부터 관리해야 한다.

- 사업자등록
- 통신판매업 신고
- 전자상거래법상 고지 의무
- 세무 신고
- 대행수수료와 물품대금 구분
- 통관 정보 관리
- 개인통관고유부호 처리
- 제품안전 규제
- 식품·화장품·의료기기 규제
- 지식재산권·가품 리스크
- 개인정보 보호
- 반품·환불 분쟁

## 2. 사업자·통신판매 관련 준비

초기 준비 항목:

```markdown
- [ ] 사업자등록 여부 확인
- [ ] 해외직구대행업 업종코드 확인
- [ ] 통신판매업 신고 필요 여부 확인
- [ ] 판매채널 가입
- [ ] 정산 계좌 준비
- [ ] 사업용 이메일 준비
- [ ] 고객 응대용 연락 채널 준비
- [ ] 개인정보 처리 기준 정리
```

## 3. 세무 리스크

### 3.1 핵심 쟁점

해외 구매대행은 일반 상품 판매와 달리 다음 구분이 중요하다.

- 고객에게 받은 전체 금액
- 실제 물품 구입비
- 해외/국내 배송비
- 관부가세
- 플랫폼 수수료
- 판매자의 대행수수료 또는 이익

세무상 매출 인식과 증빙 처리는 사업 초기부터 일관되게 기록해야 한다.

### 3.2 주문별 보관 자료

```markdown
- [ ] 고객 주문내역
- [ ] 고객 결제금액
- [ ] 해외 상품 결제내역
- [ ] 해외 배송비
- [ ] 배대지 비용
- [ ] 국제배송비
- [ ] 국내배송비
- [ ] 플랫폼 수수료
- [ ] 환불·반품 비용
- [ ] 최종 마진
- [ ] 고객 안내 내역
```

### 3.3 주의사항

- 개인 계좌와 사업 계좌를 가능한 분리한다.
- 주문별 원가와 증빙을 보관한다.
- 플랫폼 정산금만 보고 손익을 판단하지 않는다.
- 배송비와 물품대금을 대행수수료와 구분해 기록한다.
- 부업이라도 사업소득 신고 의무를 가볍게 보면 안 된다.

## 4. 통관 리스크

### 4.1 개인통관고유부호

해외 구매대행에서는 수취인 정보와 개인통관고유부호가 중요하다.

주의사항:

- 수취인명과 개인통관고유부호 정보 일치 확인
- 고객 정보 오입력 시 통관 지연 가능성 안내
- 고객 개인정보 최소 수집
- 주문 처리 후 불필요한 개인정보 보관 지양

### 4.2 판매 목적 물품과 자가사용 통관

판매 목적 물품을 개인 자가사용처럼 통관하는 방식은 리스크가 크다. 구매대행 구조와 실제 수입판매 구조가 혼동되지 않도록 해야 한다.

### 4.3 관부가세 안내

고객에게 다음을 사전에 안내해야 한다.

- 관부가세 발생 가능성
- 과세 기준
- 고객 부담 여부
- 통관 과정에서 추가 자료가 필요할 수 있음
- 여러 주문 합산 시 과세 가능성

## 6. 지식재산권 리스크

주의할 상품:

- 유명 브랜드 로고 포함
- 캐릭터 상품
- 명품 스타일 모방
- 정품 여부 불분명
- 병행수입으로 오해될 수 있는 상품
- 이미지 저작권이 불명확한 상품

대응 원칙:

- 브랜드 키워드 남용 금지
- 정품 보장 표현 남용 금지
- 원본 이미지 사용권 확인
- 가품 의심 상품 제외
- 캐릭터·IP 상품 초기 제외

## 7. 반품·환불 리스크

구매대행은 반품 비용이 크다.

사전에 정해야 할 것:

- 단순 변심 반품 가능 여부
- 해외 왕복 배송비 부담 주체
- 주문 후 취소 가능 단계
- 해외 발주 후 취소 제한 조건
- 불량·파손 판정 기준
- 사진·영상 증빙 요청 방식
- 환불 처리 기간

## 8. 개인정보 리스크

취급 정보:

- 이름
- 연락처
- 주소
- 개인통관고유부호
- 주문 상품
- 배송 추적번호

관리 원칙:

- 필요한 정보만 수집
- 접근 권한 최소화
- 시트 공유 제한
- 불필요한 장기 보관 지양
- 외부 AI 도구에 개인정보 입력 금지
- 자동화 도구 연결 시 권한 범위 확인
