# 05. 상품등록·상세페이지·CS 운영

## 1. 상품등록의 목적

상품등록은 단순 노출 작업이 아니라 다음 4가지를 동시에 수행해야 한다.

1. 검색에 걸리는 상품명과 키워드 구성
2. 고객이 구매 결정을 할 수 있는 정보 제공
3. 구매대행 특성에 따른 리스크 고지
4. 향후 CS를 줄이기 위한 사전 안내

## 2. 상품명 작성 기준

상품명에는 다음을 반영한다.

- 핵심 키워드
- 용도
- 주요 속성
- 옵션 구분
- 사이즈 또는 규격
- 불필요한 과장 표현 제외
- 브랜드 오인 가능 표현 제외
- 인증·효능 관련 과장 표현 제외

예시 구조:

```text
[핵심키워드] [용도/대상] [주요 특징] [규격/옵션]
```

## 3. 상세페이지 필수 구성

### 3.1 상단 요약

- 어떤 상품인지
- 누구에게 필요한지
- 어떤 문제를 해결하는지
- 핵심 장점 3가지

### 3.2 상품 정보

- 구성품
- 소재
- 크기
- 색상
- 옵션
- 사용 방법
- 주의사항

### 3.3 옵션 안내

옵션이 복잡하면 CS와 오배송 가능성이 커진다.

상세페이지에는 다음을 명확히 써야 한다.

- 옵션명
- 옵션 이미지
- 옵션별 차이
- 호환 가능/불가능 조건
- 사이즈 측정 기준
- 색상 차이 가능성

### 3.4 해외 구매대행 고지

필수 고지 예시:

```markdown
본 상품은 해외 판매처에서 고객 주문 후 구매가 진행되는 해외 구매대행 상품입니다. 주문 후 해외 현지 재고와 가격을 확인한 뒤 발주가 진행되며, 해외 배송·통관·국내 배송 과정을 거쳐 수령까지 일정 기간이 소요될 수 있습니다.
```

### 3.5 배송 고지

포함할 내용:

- 평균 배송 기간
- 지연 가능성
- 해외 현지 재고 변동 가능성
- 통관 지연 가능성
- 명절·세일 기간 지연 가능성
- 배송 추적 가능 시점

예시:

```markdown
해외 구매대행 특성상 평균 배송 기간은 영업일 기준 7~15일 정도 소요될 수 있으며, 해외 판매처 사정, 항공/해상 물류, 통관 상황에 따라 지연될 수 있습니다.
```

### 3.6 통관 고지

포함할 내용:

- 개인통관고유부호 필요 여부
- 수취인 정보 일치 필요
- 관부가세 발생 가능성
- 통관 보류 시 고객 협조 필요

### 3.7 반품·교환 고지

포함할 내용:

- 단순 변심 반품 가능 여부
- 해외 왕복 배송비 부담
- 개봉·사용 후 반품 제한
- 주문 후 발주 단계에서 취소 제한 가능성
- 불량·오배송 처리 기준
- 반품 소요 기간

## 4. CS 템플릿

### 4.1 주문 접수 안내

```text
안녕하세요. 주문 감사합니다.
해당 상품은 해외 구매대행 상품으로, 해외 판매처 재고와 가격 확인 후 발주가 진행됩니다.
발주 완료 후 배송 진행 상황을 순차적으로 안내드리겠습니다.
```

### 4.2 개인통관고유부호 요청

```text
해외 구매대행 상품 통관을 위해 수취인의 개인통관고유부호가 필요합니다.
주문자명, 수취인명, 연락처, 개인통관고유부호가 일치하지 않으면 통관이 지연될 수 있으니 확인 부탁드립니다.
```

### 4.3 배송 지연 안내

```text
현재 상품은 해외 배송/통관 과정에서 예상보다 지연되고 있습니다.
구체적인 진행 상태를 확인 중이며, 변동 사항이 확인되는 즉시 안내드리겠습니다.
해외 구매대행 특성상 현지 물류와 통관 상황에 따라 일정이 변동될 수 있는 점 양해 부탁드립니다.
```

### 4.4 품절 안내

```text
주문하신 상품이 해외 현지 판매처에서 품절로 확인되었습니다.
불편을 드려 죄송합니다. 원하시면 주문 취소 및 환불을 진행해드리겠습니다.
```

### 4.5 반품비 안내

```text
해당 상품은 해외 구매대행 상품으로 단순 변심 반품 시 해외 왕복 배송비 및 국내 배송비가 발생할 수 있습니다.
정확한 반품 가능 여부와 비용은 상품 상태와 해외 판매처 정책 확인 후 안내드리겠습니다.
```

## 5. P0 상세페이지 생성기의 입력값

P0에서 상세페이지를 생성하려면 다음 입력값이 필요하다.

| 입력값 | 설명 |
|---|---|
| 원본 상품 URL | 해외 상품 페이지 |
| 상품명 | 원본명 또는 임시명 |
| 카테고리 | 국내 등록 카테고리 |
| 가격 | 해외 상품가 |
| 배송비 | 현지/국제/국내 배송비 |
| 옵션 | 색상, 사이즈, 구성 등 |
| 이미지 | 대표 이미지와 상세 이미지 |
| 스펙 | 크기, 소재, 구성품 |
| 금지/주의 여부 | 규제 리스크 체크 |
| 배송 기간 | 예상 배송 기간 |
| 반품 조건 | 해외 판매처와 국내 판매 정책 |
| 타깃 고객 | 누구에게 파는지 |
| 핵심 소구점 | 왜 사야 하는지 |
| 경쟁 상품 메모 | 국내 유사 상품 비교 |

## 6. 상세페이지 생성 결과물

P0의 출력물은 다음으로 구성하는 것이 좋다.

- 상품명 후보 3개
- 검색 키워드 후보
- 상세페이지 상단 카피
- 주요 장점 3~5개
- 옵션 설명
- 스펙 표
- 구매 전 확인사항
- 해외 구매대행 고지문
- 배송/통관 고지문
- 반품/교환 고지문
- CS 예상 질문 5개
- 등록 전 체크리스트
