## 2. 상품 리스크 체크리스트

```markdown
- [ ] 식품/건기식이 아니다
- [ ] 화장품이 아니다
- [ ] 의료기기가 아니다
- [ ] 의약품이 아니다
- [ ] 전기·전자제품이 아니다
- [ ] 배터리 포함 제품이 아니다
- [ ] 어린이제품이 아니다
- [ ] 검역 대상이 아니다
- [ ] 유명 브랜드/캐릭터/IP 상품이 아니다
- [ ] 가품 의심 요소가 없다
- [ ] 액체/분말/칼날/자석 등 배송 제한 요소가 없다
- [ ] 파손 위험이 낮다
- [ ] 옵션이 복잡하지 않다
- [ ] 반품 사유가 많지 않을 상품이다
- [ ] 상세페이지 작성에 필요한 정보가 충분하다
```

## 4. 상품등록 전 체크리스트

```markdown
- [ ] 상품명이 검색 키워드를 포함한다
- [ ] 대표 이미지가 상품을 명확히 보여준다
- [ ] 옵션명이 원본과 일치한다
- [ ] 사이즈/색상 차이 가능성을 고지했다
- [ ] 해외 구매대행 상품임을 고지했다
- [ ] 배송 기간을 고지했다
- [ ] 통관 정보를 고지했다
- [ ] 개인통관고유부호 필요 여부를 안내했다
- [ ] 관부가세 발생 가능성을 안내했다
- [ ] 반품/교환 조건을 고지했다
- [ ] 품절 가능성을 고지했다
- [ ] 가격 변동 가능성을 내부적으로 확인했다
- [ ] 원가·마진 계산이 완료되었다
- [ ] 금지/주의 품목 체크를 완료했다
```

## 6. CS 유형 분류

| 유형 | 설명 | 대응 |
|---|---|---|
| 배송 문의 | 배송 위치/기간 문의 | 추적 현황 안내 |
| 배송 지연 | 예상 기간 초과 | 사전 사과 및 원인 안내 |
| 품절 | 해외 판매처 품절 | 취소/대체 제안 |
| 가격 변동 | 발주 전 가격 상승 | 취소 또는 추가비용 정책 판단 |
| 옵션 오류 | 고객 선택/등록 오류 | 증빙 확인 후 처리 |
| 파손 | 배송 중 파손 | 사진 요청, 배대지/판매처 정책 확인 |
| 불량 | 상품 자체 문제 | 증빙 요청 후 환불/교환 판단 |
| 단순 변심 | 고객 변심 | 반품 가능 여부와 비용 안내 |
| 통관 지연 | 정보 오류/검사 | 고객 정보 확인 및 진행 안내 |

## 7. 상세페이지 프롬프트 템플릿

```markdown
다음 해외 구매대행 상품 정보를 바탕으로 국내 오픈마켓용 상세페이지 초안을 작성해줘.

조건:
- 과장 광고를 피한다.
- 해외 구매대행 상품임을 명확히 고지한다.
- 배송/통관/반품 안내를 포함한다.
- 고객 문의가 줄어들도록 구매 전 확인사항을 작성한다.
- 상품명 후보, 핵심 소구점, 상세페이지 본문, 스펙표, 옵션 안내, CS 예상 질문을 분리해서 작성한다.

상품 정보:
- 원본 URL:
- 상품명:
- 카테고리:
- 가격:
- 옵션:
- 소재/크기/스펙:
- 타깃 고객:
- 핵심 장점:
- 주의사항:
- 예상 배송 기간:
- 반품 조건:
```

## 8. 상품 리스크 검토 프롬프트 템플릿

```markdown
다음 상품이 해외 구매대행 초보자가 취급하기에 적합한지 검토해줘.

검토 기준:
- 식품/건기식/화장품/의료기기/의약품 여부
- 전기·전자/KC 가능성
- 어린이제품 가능성
- 브랜드/IP/가품 리스크
- 배송 제한 요소
- 파손/반품/CS 리스크
- 옵션 복잡도
- 부업형 운영 적합성

출력:
1. 종합 판정: A/B/C/D
2. 주요 리스크
3. 추가 확인이 필요한 항목
4. 초보자 취급 여부
5. 상세페이지에 반드시 고지할 내용

상품 정보:
- 상품명:
- 원본 URL:
- 카테고리:
- 이미지 설명:
- 주요 스펙:
- 옵션:
```

## 10. 프로젝트 미해결 질문

```markdown
- 어떤 소싱처를 첫 번째로 사용할 것인가?

- 첫 판매채널은 스마트스토어로 할 것인가, 다른 오픈마켓으로 할 것인가?

- 규제 리스크 판단은 체크리스트형으로 둘 것인가, 점수형으로 둘 것인가?
```
