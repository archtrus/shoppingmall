# 해외 구매대행 부업 프로젝트 소스 문서

## 목적

이 문서 세트는 해외 구매대행을 **주업이 아닌 부업**으로 운영하면서, 동시에 향후 **상세페이지 생성 AI / 구매대행 운영 보조 서비스**를 개발하기 위한 프로젝트 소스로 사용하기 위해 정리한 것이다.

4개의 리서치 보고서에서 중복되는 내용을 제거하고, 이후 기획·Codex 구현·운영 SOP 작성에 바로 재사용할 수 있도록 주제별 Markdown 파일로 분리했다.

## 핵심 결론

해외 구매대행 부업의 핵심은 “상품을 많이 올리는 것”이 아니라 다음 5가지를 작게 검증하는 것이다.

1. 규제 리스크가 낮은 카테고리만 선별한다.
2. 주문 후 발주·배대지·통관·국내배송·CS를 끊기지 않게 운영한다.
3. 원가·마진·수수료·환율·배송비를 주문 단위로 기록한다.
4. 반품·지연·품절·통관 문제에 대한 표준 응대 템플릿을 만든다.
5. 외부 툴과 AI 자동화는 “운영이 먼저 정리된 뒤” 적용한다.

## 현재 프로젝트에서의 사용법

이 문서 세트는 다음 3가지 용도로 활용한다.

- 실제 부업 운영 SOP
- 구매대행 서비스 / 상세페이지 생성 AI의 요구사항 정의서
- Codex 작업용 프롬프트와 기능 명세의 원천 자료

## 중요한 전제

이 문서는 사업 판단과 기획을 위한 내부 작업물이다. 실제 사업자등록, 통신판매업 신고, 세무신고, 품목별 인허가, KC·식약처·관세 이슈는 최신 공식 자료와 전문가 검토가 필요하다.
